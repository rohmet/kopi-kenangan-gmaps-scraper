# Kopi Kenangan GMaps Scraper

A tool for scraping Google Maps reviews for Kopi Kenangan outlets.

## Contents
- `scrap_data_reviews.py`: The main scraper script.
- `data-scrap/`: Directory containing scraped reviews in CSV format.

## Setup
1. Install dependencies (e.g., Selenium, Pandas).
2. Run the script:
   ```bash
   python scrap_data_reviews.py
   ```
